# Git 명령어 정리

## 커밋 기록 확인

- git log

git log 종료 하기
 - 커맨드라인에서 키보드'Q' press


git 설정
 git config --global user.email "email@mail.com"
 git config --global user.name "gildong"


git init
- 해당 디렉토리를 git으로 관리하기 시작하도록 초기화 설정

git status
- 깃 상태 확인
- 빨강 표시: 언트래킹
- 녹색 표시: 트래킹

git add *
 - 깃 상태 트래킹 on
 - 아직 커밋 이전

git commit -m "test.txt 추가"
 - 로컬 -> 

git branch -M main
 - 브랜치 생성

git branch -vv
 - 현재 브랜치 확인
