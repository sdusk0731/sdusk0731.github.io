# 기본 동작 원리
```
Working Directory
 - 작업하는 파일이 있는 디렉토리

Staging Area
 - Git에 등록할 (커밋) 파일들이 올라가는 영역

Local Repository (로컬))
 - 로컬 Git프로젝트의 메타데이터와 데이터 정보가 저정되는 영역

Remote Repository (깃허브 서버)
 - Github등의 서비스를 통한 온라인 상의 저장소
```
