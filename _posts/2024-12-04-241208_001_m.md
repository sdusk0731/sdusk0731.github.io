# 3주차 멘토링: 면접 팁

## 면접 TIP
```
면접 = 면접관이 질문을 지원자에게 하는 것

1. 면접이 어려운 이유 = 어떤 질문을 받을지 몰라서 어렵다.
 -> 어떤 질문을 받을지 미리 안다면?
 -> 그 질문은 어디서 나오지?
 -> 작성한 자소서 혹은 나의 답변
 -> 면접관이 할 수 있는 질문의 폭을 줄이자. (적은 키워드를 보여줘야 질문이 적다)
 -> 하나의 키워드로 스토리 라인 작성

2. 면접관 입장에서의 질문 = 자소서를 읽고 질문
 => 자신의 자소서를 읽고 자문 자답 해볼것.

3. 면접 질문 N : 면접 답변 1 = 좋은 면접 준비

4. 내가 준비한 내용을 질문 하도록 공격적으로 면접흐름을 이끌어 가야 한다. (공격형 면접)

5. 10개의 질문에 BFS로 대답하는것 보다, 1개라도 "DFS"로 대답하는게 승산이 있다.
EX) Q.사용할 줄 아는 언어는?
 ### BFS : 이것 저것 등등 사용할 줄 알아요.
 ### DFS : JAVA에 대한 깊은 애용에 대해 답변

* DFS (깉이 우선 탐색)
* BFS (넓이 우선 탐색)
```
