# 인기글 서비스 개선

![서술형문제](https://sdusk0731.github.io/assets/images/20250112_001_m/서술형문제.png)

---
### 인기글 선정 기준
```
 - 좋아요 수가 많음
 - 조회수가 높음
 - 글자 수가 많음
 - 오래된 글
 ```


### 개선 필요 사항
```
1. 서비스가 느려짐
2. 5분마다 계산할 필요 없음
```

### 개선 방안
```
1. 5분마다 인기글 계산 로직을 수행하고 캐시에 저장하여 호출한다.
 > 게시글id를 캐시에 저장
2. 인기글은 별도의 구분자를 두어 따로 표시할 수 있도록 한다.
```
