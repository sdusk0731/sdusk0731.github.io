# Fork복제

## Fork란?
```
- 다른사람의 원격 저장소에서 어떤 부분을 수정하거나 추가 기능을 넣고 싶을때 
 해당 원격 저장소를 내 원격 저장소에 그대로 복제하는 것, 전문용어로 "포크뜬다"라고 한다.
```

## Fork복제
```
- 다른 계정으로 접속
- 원하는 레포지토리에 Github페이지 접속 후 Fork 버튼 클릭
```
